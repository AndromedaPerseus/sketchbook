Magnetic Pointer

This demo will make use of CSS anchor positioning if it's available. It should work either way. You'll need to be in Chrome Canary with the "Experimental Web Platform Features" flag if you want to use anchor positioning.

To make this demo work, add a "data-anchor" attribute to things you want the cursor to stick to!
